# 项目全生命周期

## 启动与授权

- [ ] 核验合同、服务范围、授权书、测试时间窗、联系人、紧急停止机制与数据处理要求。
- [ ] 明确目标资产、排除范围、生产影响约束、变更要求、沟通节奏和验收标准。

## 调研与方案

- [ ] 收集业务架构、资产清单、网络拓扑、账号权限、日志、历史问题和现有控制措施。
- [ ] 输出实施方案：方法、人员、排期、风险控制、数据需求、交付物与依赖项。

## 实施与质量

- [ ] 按计划执行，记录操作、时间、证据来源、影响评估和异常处置。
- [ ] 执行每日/阶段同步、内部复核、客户确认和变更留痕。

## 交付与闭环

- [ ] 交付报告、问题清单、原始证据说明、整改建议、复测计划与管理层摘要。
- [ ] 组织汇报、答疑、验收、整改跟踪和项目复盘。
