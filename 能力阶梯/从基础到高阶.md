# 从基础能力向高阶能力跃迁

## 一级：基础技术储备

- [ ] 掌握网络、操作系统、身份认证、数据库、Web、脚本与日志的基础原理。
- [ ] 能在隔离实验环境中完成资产识别、配置检查、日志检索和基础漏洞验证。
- [ ] 能读懂架构图、流量、日志、配置与源代码片段，并清晰记录证据。

## 二级：单项服务可交付

- [ ] 按授权范围独立完成一种服务的准备、实施、报告和复测。
- [ ] 能解释风险形成条件、影响范围、整改优先级与验证方法。
- [ ] 能使用标准模板与质量清单交付可验收成果。

## 三级：跨域综合分析

- [ ] 将资产、漏洞、身份、网络、日志和业务流程关联为攻击面与风险路径。
- [ ] 能为复杂环境设计测试方案、证据链和分阶段整改路线。
- [ ] 能把单次发现转化为检测规则、运营指标或工程改进项。

## 四级：高阶对抗与体系建设

- [ ] 能开展以目标为导向、受控且可回滚的攻防验证与桌面/实战演练。
- [ ] 能设计威胁建模、检测工程、自动化运营和重大保障体系。
- [ ] 能基于业务目标量化风险，推动安全架构和组织流程持续改进。

## 成长证据待办

- [ ] 为每一级建立技能矩阵、案例清单、复盘记录和评审标准。
- [ ] 建立“学习—实验—陪同交付—独立交付—复盘授课”的培养闭环。
- [ ] 制定季度能力盘点与薄弱项补齐计划。
