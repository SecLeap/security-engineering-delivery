# 服务能力地图

## 服务域

| 服务域 | 服务项 | 核心产出 |
| --- | --- | --- |
| 基础评估与验证 | 巡检、漏洞扫描、渗透测试、边界/设备有效性、域控检查 | 问题清单、验证证据、整改建议、复测结论 |
| 资产与暴露面 | 公网资产/数据暴露、内网资产、攻击路径测绘 | 资产底账、暴露面、路径图、优先级 |
| 攻防演练 | 红队、护网防守、应急演练、钓鱼演练 | 演练方案、过程记录、检测与处置改进 |
| 运营与应急 | 重保、应急响应、运营支撑、告警降噪、情报共享 | 值守记录、事件闭环、运营指标、检测规则 |
| 咨询与能力建设 | 培训、驻场、威胁建模 | 能力评估、培训材料、驻场成果、模型与路线图 |

## 共用能力依赖

- [ ] 将每项服务映射到资产识别、漏洞管理、日志检测、身份权限、网络边界、云与应用安全能力。
- [ ] 梳理可复用的访谈提纲、数据采集项、证据清单、报告章节、验收标准。
- [ ] 建立服务之间的衔接规则，例如测绘→评估→整改→复测→运营监测。
